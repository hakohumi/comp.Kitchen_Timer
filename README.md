2020/07/02

- 作業予定
  - [ ] フローチャート 作成

- 長押しの定義
  - スイッチを押してから、1秒経つ
    - スイッチの立ち上がりから、1秒が経過する
      - スイッチの立ち上がりで、長押しフラグを立てる
      - 1秒経過するまで、長押しフラグを確認する
        - 1秒が経過したら、長押し状態をONにする
  - 要素
    - 長押しフラグ
      - 長押し状態になるまでの時間で、押され続けていることを比較する
    - 長押し状態
      - スイッチを押し続けて、1秒経ったらONになる
        - ONの挙動：
          - カウント単位が増える
        - OFFへの条件
